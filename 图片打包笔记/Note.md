3. 用vscode的图形界面
   
    首先，点击源代码管理项目窗口，点击对应文件的暂存更改。对应的是命令行的git add 命令

    其次，点击提交，输入相应的信息，对应的是命令行的，git commit -m 命令

    最后，点击右上角的3个省略号，选择推送。对应的是命令行的，git push 指定网址 指定文件
